O Problema

Empresas dizem uma coisa.
Fazem outra.

Relatorios anuais, earnings calls, investor days — todos projetam uma narrativa estrategica coerente. Mas entre o discurso e a execucao, surgem lacunas que investidores e conselhos muitas vezes so percebem tarde demais.

⚠

Drift Estrategico

A empresa declara "foco em fintech" mas aloca 60% do capex em hardware. As acoes contradizem os pilares estrategicos.

⚖

Posicionamento Cego

Sem benchmark sistematico contra pares, e impossivel saber se a empresa esta liderando ou ficando para tras no setor.

💹

Drivers Ocultos

O que realmente move o multiplo de valuation? Sem analise de correlacao, gestores operam no escuro sobre o que o mercado precifica.

A Solucao

Tres pipelines de analise.
Um sistema integrado.

Cada pipeline responde a uma pergunta fundamental de governanca corporativa. Juntos, formam uma visao completa da saude estrategica de uma empresa.

01
DRIFT DETECTION

A empresa faz o que diz?

Extrai pilares estrategicos declarados, mapeia acoes concretas, e pontua a coerencia entre discurso e execucao em 5 dimensoes.

Fontes → Coleta → Pilares → Mapeamento → Score → Relatorio
02
PEER COMPARISON

Como se compara aos pares?

Identifica concorrentes, padroniza metricas, e gera rankings comparativos com contexto estrategico para cada posicao.

Pares → Metricas → Coleta → Padronizacao → Rankings → Relatorio
03
VALUATION DRIVERS

O que move o multiplo?

Analisa um universo de 20+ empresas para descobrir quais metricas e acoes realmente impulsionam a valorizacao de mercado.

Industria → Dados → Correlacao → Deep Dives → Playbook

Classificacao de Drift

O pipeline de drift gera um score de coerencia de 0 a 5 por pilar estrategico, ponderando 5 dimensoes.

≥ 4.0
Alinhado
Acoes consistentes com a estrategia
3.0–3.9
Drift Menor
Desvios pontuais e gerenciaveis
2.0–2.9
Drift Significativo
Contradicoes entre discurso e acao
< 2.0
Desconexao
Estrategia declarada e irrelevante
Os Agentes

13 agentes especializados.
Coordenados por IA.

Nao e um unico modelo fazendo tudo. Cada agente tem escopo definido, fontes permitidas, e formato de output rigido. O orquestrador distribui tarefas em paralelo e valida cada entrega.

industry-scanner
Industry Scanner
Identifica o universo de empresas comparaveis e define o perimetro da analise.
source-cataloger
Source Cataloger
Cataloga fontes publicas com tags de vies: filing, analyst, industry-report.
metrics-designer
Metrics Designer
Define a taxonomia de metricas relevantes para o setor e suas formulas.
data-collector
Data Collector ×3
Tres tiers paralelos coletando dados quantitativos de ~9 empresas cada.
strategy-researcher
Strategy Researcher
Extrai perfis estrategicos e acoes concretas de cada empresa do universo.
statistical-analyst
Statistical Analyst
Padroniza dados, calcula correlacoes Spearman, ranqueia value drivers.
platform-profiler
Platform Profiler
Deep-dives em cada plataforma: modelo operacional, vantagens, riscos.
sector-specialist
Sector Specialist
Analise por classe de ativo e vertical de mercado.
insight-synthesizer
Insight Synthesizer
Consolida todas as evidencias em plays acionaveis e anti-patterns.
report-composer
Report Composer
Gera o relatorio final em HTML seguindo o SIGMA writing guide.
target-lens
Target Company Lens
Extrai principios transferiveis para a empresa-alvo, com cascata de governanca.
fact-checker
Fact Checker
3 checkpoints de auditoria. Claims sem evidencia = bloqueio imediato.

Para Desenvolvedores

Arquitetura, stack, e como rodar.

Stack Principal

  • Orquestracao Claude Code Agent teams
  • LLM Claude Sonnet via Anthropic SDK
  • Backend Python 3.11+ com type hints
  • Desktop Tauri (Rust + React + TypeScript)
  • Styling Tailwind CSS
  • Terminal xterm.js + portable-pty
  • Fontes DM Sans + IBM Plex Mono (woff2)

Principios de Design

  • Prompts Version-controlled em prompts/
  • Evidencia Toda claim cita fonte rastreavel
  • Anti-hallucination Fact Checker em 3 checkpoints
  • Bias tags Toda fonte recebe classificacao
  • Tone Academico, sem marketing speak
  • Iteracao --base-run para refinar runs
  • Delta Nunca re-coleta dados existentes

Quick Start

# Instalar dependencias Python
pip install -r requirements.txt
 
# Rodar o dashboard desktop
cd src/tauri && npm install && npm run tauri dev
 
# Executar pipeline VDA via CLI
/valuation-driver SIGMA --auto
 
# Executar drift detection
/analyze-drift SIGMA
 
# Checar qualidade dos dados
python3 -m src.analyzer.data_gaps --run-dir data/processed/pax/2026-03-09-run2/
